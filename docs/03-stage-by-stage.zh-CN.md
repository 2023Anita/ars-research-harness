# 03. S0-S9 分阶段说明

| 阶段 | 关键问题 | 不允许做什么 |
|---|---|---|
| S0 | 目标是什么，边界是什么？ | 不直接写论文 |
| S1 | 研究问题是否可回答？ | 不直接跑分析 |
| S2 | 数据、变量、模型和敏感性分析怎么定？ | 不临时改口径 |
| S3 | 按方案执行分析 | 不过度解释 |
| S3b | 模型是否需要简化？ | 不硬塞过拟合模型 |
| S4 | 结果意味着什么？ | 不做因果夸大 |
| S5 | IMRaD 结构如何组织？ | 不写完整正文 |
| S5b | 文献矩阵是否够支撑论点？ | 不伪造引用 |
| S6 | 写完整初稿 | 不声称已投稿 |
| S7 | 核查数据、引用、论断 | 不忽略风险 |
| S7b | 清理引用和参考文献 | 不保留未核验占位文献 |
| S8 | 模拟审稿和修回路线 | 不直接定稿 |
| S8b | 实施修回 | 不跳过表图和统计细节 |
| S9 | 生成投稿前包 | 不承诺期刊接收 |

每一阶段都应该有一个 checkpoint 文件，并在 `workflow-run.json` 里记录状态。
